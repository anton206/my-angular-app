# jak utworzyć projekt w Angularze?

## 1. zainstaluj angular cli
```
npm install -g @angular/cli
```
jak masz admina to PATH sam sie ustawi

## 2. stwórz projekt
```
ng new moj-projekt
```

## 3. uruchom projekt
```
cd moj-projekt
npm start
```

projekt bedzie dostepny na localhost:4200
